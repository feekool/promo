Обновления навсегда

Базовая Open Source версия с репозитория получает все актуальные патчи. Вы в тренде без лишних затрат.

📦 Установка за 1 минуту — это реально
1
Зарегистрируйтесь на GitHub — если у вас ещё нет аккаунта, создайте бесплатно на github.com.
2
Перейдите в репозиторий-шаблон
https://github.com/feekool/feekool.github.io — официальная базовая Open Source версия.
3
Нажмите кнопку "Use this template" → "Create a new repository". Назовите его, например, "my-forum".
4
Включите GitHub Pages: перейдите в Settings → Pages → выберите ветку main (или master) → Save. Через минуту ваш форум уже онлайн!
5
Готово! 🎉 Ваш персональный форум работает. Вносите изменения, добавляйте разделы и меняйте оформление под любую идею. Все обновления базовой версии приходят автоматически (синхронизируйте форк).
💡 Альтернативный способ: Скачайте движок архивом и установите на любой другой хостинг (поддерживается статический HTML). Но самый простой путь — GitHub Pages.
