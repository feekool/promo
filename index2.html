📦 Установка за 1 минуту

  1. 1
    Зарегистрируйтесь на GitHub Создайте бесплатный аккаунт на github.com
  2. 2
    Откройте шаблон
    github.com/feekool/feekool.github.io
  3. 3
    Нажмите "Use this template" → "Create a new repository". Назовите его, например, "my-forum"
  4. 4
    Включите GitHub Pages: Settings → Pages → ветка main → Save. Через минуту форум онлайн!
  5. 5
    Готово! 🎉 Ваш форум работает. Меняйте оформление, добавляйте разделы — всё под вашим контролем.
💡 Альтернатива: Скачайте архив и установите на любой хостинг со статическим HTML. Но GitHub Pages — самый простой путь.
